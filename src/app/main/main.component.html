Bienvenue

Veggie Time est une application vous permmettant d'afficher des recettes (végétariennes) aléatoires.

Pour commencer, faites un tour sur les pages suivantes :

  • liste de recettes
  • liste d'ingrédients
  • formulaire de contact

Usage de l'API limité à 50 requêtes / jour.

Ce projet est réalisé dans le cadre du cours Développement mobile web en 3ème année de licence informatique à l'ISFATES par Marie-Lou Wechsler.
